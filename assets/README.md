# Assets Folder

This folder contains resources for the Egg Timer app.

## Required Files

### icon.png
- App icon (recommended size: 512x512 pixels)
- You can create one using:
  - Online tools like Canva or Figma
  - Use an egg emoji screenshot
  - Download a free icon from icons8.com or flaticon.com

### alarm.mp3 (optional)
- Sound that plays when timer completes
- If not provided, the app will generate a beep sound
- You can use any short MP3 sound file
